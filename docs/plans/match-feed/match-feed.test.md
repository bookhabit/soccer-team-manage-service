# match-feed 기능 테스트 가이드

> seed 실행: `cd server && npm run seed:match-feed`  
> 전제: `seed:club`, `seed:match` 가 먼저 실행되어 있어야 합니다.

---

## 시드 데이터 요약

| 클럽 | 소재지 | 기록 완료 경기 | 타입 |
|------|--------|----------------|------|
| 마무리FC | 서울특별시 종로구 | 10개 (기존 2 + 신규 8) | LEAGUE 7 / SELF 3 |
| 카동FC | 서울특별시 강남구 | 5개 | LEAGUE 3 / SELF 2 |
| 부산FC | 부산광역시 해운대구 | 5개 | LEAGUE 3 / SELF 2 |
| **합계** | | **20개** | 피드 1페이지 꽉 참 |

**피드 노출 제외 데이터** (검증용):
- `삭제된 경기` (isDeleted=true) — 피드에 보이면 안 됨
- `기록 미제출 경기` (isRecordSubmitted=false) — 피드에 보이면 안 됨
