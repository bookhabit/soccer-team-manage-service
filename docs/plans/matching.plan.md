# 경기 매칭 기획안

## 개요

팀 단위로 상대팀을 모집하거나 탐색하는 기능.  
용병 탭과 별개인 **독립 탭**으로 제공한다.  
등록·수락 권한은 클럽 관리자(주장·부주장)에게만 부여한다.

---

## 매칭 목록 페이지

**게시글 1개 카드 데이터**

- 팀 정보 (이름, 레벨, 대표 이미지)
- 날짜 / 요일 / 시간
- 인원 (예: 11 vs 11)
- 성별 (혼성 / 남성 / 여성)
- 구장 장소
- 실력 레벨
- 참가비
- 매칭 상태 (모집 중 / 매칭 완료 / 만료)

**검색 및 필터**

- 날짜 범위
- 지역
- 실력 레벨
- 성별
- 참가비 유무

---

## 매칭 상세 페이지

- 팀 정보 (이름, 레벨, 대표 이미지 — 클릭 시 클럽 프로필 이동)
- 날짜 / 요일 / 시간
- 인원
- 성별
- 구장 (이름 + 주소)
- 실력 레벨
- 참가비
- 작성일
- 매칭 상태 표시 (모집 중 / 매칭 완료 / 만료)
- **매칭 신청 버튼** (상대팀 관리자에게만 표시, 본인 팀 게시글엔 미표시)

---

## 매칭 등록 페이지

> 권한: 클럽 관리자(주장·부주장)만 등록 가능

- 팀 정보 (본인 소속팀 자동 채움)
- 날짜 / 요일 / 시간
- 인원 선택
- 성별 선택 (혼성 / 남성 / 여성)
- 구장 입력 (이름 + 주소)
- 실력 레벨 선택
- 참가비 (무료 / 금액 직접 입력)
- 문의용 연락처 (기본값: 등록자 휴대폰)

**수정/삭제**

- 등록자(관리자)만 가능
- 매칭 완료 상태에서는 수정 불가, 삭제만 가능

---

## 매칭 공유 기능

- 등록된 매칭 게시글을 팀원에게 공유
- 목적: "이 상대 어때요?" 내부 의견 수렴
- 공유 방식: 앱 내 알림 + 링크 복사

---

## 매칭 성사 플로우

### 신청

1. 상대팀 관리자가 매칭 상세 페이지에서 **"매칭 신청"** 버튼 클릭
2. 신청 메시지 입력 (선택)
3. 서버에서 게시글 등록팀 관리자에게 푸시 알림 발송
   - 신청팀 이름 / 레벨 / 신청 메시지 포함

### 수락/거절

4. 등록팀 관리자: **수락 / 거절** 버튼
5. **수락 시**
   - 양측 관리자에게 상대방 연락처(이름 + 휴대폰 번호) **인앱 알림 + 화면 표시**
   - 외부 연락(전화·문자)으로 세부 조율 가능
   - 게시글 상태 → **매칭 완료** (목록에서 완료 표시, 추가 신청 불가)
6. **거절 시**
   - 신청팀 관리자에게 거절 알림

### 연락처 노출 정책

- 수락 전까지 양측 휴대폰 번호 미노출
- 수락 이후 **인앱 화면에서 직접 표시** (이름 + 휴대폰 번호)
- 별도 인앱 채팅 없음 — 외부 앱(전화·문자)으로 연결

---

## 게시글 상태 및 만료

| 상태 | 조건 |
|---|---|
| 모집 중 | 등록 후 경기 날짜 이전, 수락 전 |
| 매칭 완료 | 수락 확정된 상태 |
| 만료 | 경기 날짜 경과 시 **자동 만료** |

- 자동 만료: 경기 날짜가 지나면 서버에서 상태 자동 변경
- 수동 삭제: 등록자(관리자)가 언제든 삭제 가능
- 만료·완료 게시글은 목록 기본 뷰에서 숨김, 필터로 조회 가능

---

## 확정 사항

| 항목 | 결정 내용 |
|---|---|
| 탭 위치 | 경기 매칭 **독립 탭** (용병 탭과 분리) |
| 게시글 만료 | 경기 날짜 경과 시 **자동 만료** + 등록자 **수동 삭제** 둘 다 지원 |
| 게시글 수정/삭제 | **등록자(관리자)만** 가능, 매칭 완료 후 수정 불가 |
| 매칭 성사 확정 | **상대팀 관리자 수락 버튼** 플로우, 수락 시 양측 연락처 인앱 공유 |
