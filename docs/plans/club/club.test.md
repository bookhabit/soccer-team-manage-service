## 🚩 시나리오 2: 클럽 생성 플로우

**[CASE ID: CLUB-04-05 ~ 09, CLUB-05-01]**

### 준비물

- 클럽 미소속 유저 (`newbie@test.com`)

### 테스트 순서

1. `newbie@test.com`으로 로그인 → 클럽 탭 → **"내 클럽 만들기"** 클릭
2. **[1단계 — 유효성 검사]**
   - 팀 이름 **빈 값** 상태에서 "다음" 클릭
   - 최대 인원 **"1"** 입력 후 "다음" 클릭
   - 최대 인원 **"51"** 입력 후 "다음" 클릭
3. **[1단계 — 정상 입력]** 팀 이름 `"FC 테스트"`, 최대 인원 `"15"` 입력 → "다음"
4. **[2단계]** 지역 `"서울 종로구"` 선택 → "다음"
5. **[3단계]** 레벨 `"AMATEUR"` 선택 → "생성하기" 클릭

### 체크포인트

- [ ] 1단계: 팀 이름 빈 값 → `"다음"` 버튼이 비활성화되거나 에러 메시지가 표시되는가?
- [ ] 1단계: 최대 인원 1 → `"2명 이상 입력하세요"` 에러가 표시되는가?
- [ ] 1단계: 최대 인원 51 → `"50명 이하 입력하세요"` 에러가 표시되는가?
- [ ] 생성 성공 후 클럽 탭 홈(대시보드)으로 자동 이동하는가?
- [ ] 생성 직후 초대 코드 화면이 노출되는가?
- [ ] 내 역할이 **"CAPTAIN"**으로 표시되는가?

## 🚩 시나리오 3: 초대 코드 입력 — 정상/예외

**[CASE ID: CLUB-04-15, 16, 17, 18, CLUB-03-20 ~ 23]**

### 준비물

- 클럽 미소속 유저 (`newbie@test.com`)

### 테스트 순서

**① 만료된 코드 입력**

1. `newbie@test.com`으로 로그인 → 클럽 탭 → **"팀 찾기"** → **"초대 코드 입력"**
2. `MAMU-EXPIRED` 입력 후 제출

**② 존재하지 않는 코드 입력**

1. `INVALID-CODE` 입력 후 제출

1. `MAMU-VALID` 입력 후 제출

### 체크포인트

> **주의**: `MAMU-VALID` 코드로 신청 완료 후 시나리오 4에서 이 신청을 승인합니다.
