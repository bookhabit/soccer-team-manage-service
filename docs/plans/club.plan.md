# 클럽 기획안

## 핵심 정책

- **1인 1팀**: 유저는 하나의 팀에만 동시 소속 가능
- **등번호**: `ClubMember` 테이블에 저장, 팀 내 unique 제약
- **클럽 평점**: 소속 팀원 전체 매너 점수의 평균값
- **지역 데이터**: 행정구역 Seed Data 방식 (공공데이터포털 → Prisma seed.ts)

---

## 클럽 생성

Funnel UI (3단계)

### 1단계

- 팀 이름
- 지역 (Region Seed Data API → Select Box)
- 레벨

### 2단계

- 팀 소개글
- 팀 로고 (대표 이미지)

### 3단계 (UI 우선 구현, 초대 기능 재구현 예정)

- 팀 생성 시 팀원 자동 초대 (선택)
  - 팀원 이름 및 연락처 입력
  - 나중에 추가하기
- 초대 링크 (복사 기능)

## 클럽 검색 페이지

### 검색 기능

- 클럽 이름 검색
- 지역 필터 (Region Seed Data API → Select Box)
- **내 주변 5km** 또는 **선호 지역** 기반 검색

> 선호 지역(`preferredRegion`)은 온보딩에서 수집

### 검색 결과 항목

- 이름 / 레벨 / 평점 / 주 장소 / 총 인원 / 소개글

---

## 가입 신청 페이지

### 신청자 정보 표시 (읽기 전용)

- 이름 / 나이 / 주 포지션 / 거주 위치 / 레벨
- 용병 경험 있는 경우: 매너 점수, 경기 횟수

### 신청 내용 입력

### 중복 신청 방지

- 서버 측 `(userId, clubId)` Unique 제약
- 신청 중인 경우 버튼 → **"신청 취소"** 로 변경
- 신청 취소 시 DB에서 삭제

- 용병 경험 있는 경우: 매너 점수, 경기 횟수

## 팀원 목록 조회 페이지

- 목록 조회
- 검색: 이름 / 포지션별 카테고리

### 팀원 1명 데이터

**개인정보** (공개·비공개 설정)

- 이름 / 전화번호 / 가입일자

**축구 정보**

- 등번호 (팀 내 unique)
- 선호 포지션 (주 포지션 + 배열) / 주 발

**경기 정보**

- 경기 수 / 골 / 도움 / MVP 횟수

**역할** — 주장 / 부주장 / 총무 등

## 팀원 상세 조회 페이지

### 화면 특이사항

- FIFA 스타일 카드 형식
  - 진입 시 카드 한 바퀴 회전 + 빛 이펙트
  - 터치 시 full화면 확대 (둥둥 떠있는 효과)
- 프로필 이미지: AI 자동 변환
- 능력치 애니메이션 / 스크롤 페이드인

### 능력치

- 속도 / 슛 / 패스 / 드리블 / 수비 / 피지컬 / 총 능력치
- **최초 입력**: 본인
- **이후 수정**: 관리자(주장·부주장)만 가능
- 미입력 또는 비공개 처리 가능

## 클럽 게시판

### 목록 조회

- 탑탭: 전체 / 공지사항 / 문의
- 게시글 데이터: NEW 뱃지 / 유형 / 제목 / 내용 2줄 / 작성자 / 작성일시 / 조회수 / 댓글 수

### 게시글 작성

- 유형: 공지사항(관리자) / 일반 / 문의
- 제목 / 내용 / 상단 고정 스위치 / 팀원 알림 전송 스위치

### 게시글 수정·삭제 권한

- **수정**: 작성자 본인만
- **삭제**: 작성자 본인 + 관리자(주장)

### 게시글 상세 조회

- 게시글 전체 내용
- 댓글 작성 / 목록 조회 (추가 기획 필요)

## 권한 이전 및 주장 탈퇴 플로우

### 주장이 팀에서 나가려면

1. 설정 → 권한 이전 페이지 진입
2. 팀원 중 1명을 새 주장으로 지정
3. 권한 이전 확정 후 탈퇴 가능

> 권한 이전 없이 탈퇴 버튼 비활성화

---

## 팀 삭제 및 해체

> 권한: 주장만 가능

### 해체 플로우

1. 주장이 "팀 해체" 요청
2. 전체 팀원에게 동의 요청 알림 발송
3. **모든 팀원 동의** 시 해체 진행
4. 팀원 1명이라도 거절 시 해체 취소

### 데이터 보존 방침

- 다른 팀과의 과거 경기 기록 보존
- 상대팀 표시: **(해체한 팀)** 으로 표기
- 팀원 개인 경기 기록(골·어시스트·MVP)은 익명화 유지

## 클럽 초대

### 초대 링크 / 초대 코드

- 유효기간: **7일**
- 만료 후 재발급 가능 (관리자만)
- 초대 링크: 웹 `/invite` 라우트 + 유니버셜 링크
- 초대 코드: 앱 내 직접 입력

---

## 강퇴

> 권한: 주장·부주장

- 팀원 목록 조회 → 1명 선택 → 강퇴 버튼
- 강퇴 이력 `ClubBanRecord` 테이블에 저장
- **강퇴된 유저는 해당 팀 재가입 불가** (서버에서 블랙리스트 체크)

## 클럽 나가기

- 내 활동 기록 (경기 수 / 골 / 포지션 / 가입~오늘 기간)
- 주의 사항 표시
- 탈퇴 사유 선택 (필수)
  - 시간이 맞지 않아요 / 다른 팀으로 이동 / 축구를 그만둘 예정 / 팀 분위기가 맞지 않아요 / 기타
- 동의 체크박스 / 팀 나가기 버튼 / 취소 버튼
