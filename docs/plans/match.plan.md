# 경기 관리 기획안

## 경기 상태 전환 기준

시간 기반 자동 전환. 별도 관리자 수동 조작 없음.

| 상태 | 조건 |
|---|---|
| 경기 전 | 현재 시각 < 경기 시작 시각 |
| 경기 중 | 경기 시작 시각 ≤ 현재 시각 ≤ 경기 종료 시각 |
| 경기 후 | 현재 시각 > 경기 종료 시각 |

> 경기 등록 시 시작 시각 + 종료 시각 모두 입력 필요

---

## 바텀탭 — 투표 탭

### 경기 투표 목록 페이지

**투표 데이터**

- 날짜 및 시간 / 요일
- 제목 / 경기 장소 / 투표 마감일
- 투표 현황 (총인원, 프로그레스바 UI)
- 투표 버튼: 참석 / 불참 / 미정

**투표 마감 후**

- 마감 후 투표 변경 불가
- 관리자만 예외적 수동 처리 가능 (개별 요청 대응)

**투표 현황 (투표 완료 후 열람)**

- 참석 / 미정 / 불참 목록

### 경기 투표 등록 페이지

> 권한: 관리자(주장·부주장) 전용

- 경기 유형: 매칭전 / 자체전
- 제목 / 날짜 / 시작 시간 / **종료 시간** / 장소 / 마감일
- 알림 전송 기능

### 경기 투표 수정·삭제

- 마감 전에만 관리자 수정 가능
- 마감 후 수정 불가, 삭제만 가능

## 포지션 배정 페이지

### 참여 선수 목록 연동

- 투표 **참석 응답자 자동 로드**
- 수동 추가 가능 (참석 미투표자 직접 추가)

### 매칭전 포지션 배정

- 쿼터 수 지정 (최대 6)
- 쿼터별 포메이션 선택
- 쿼터별 포지션 배정

### 자체전 포지션 배정

- 팀 나누기 (A팀/B팀, N팀까지 동적 생성)
- 쿼터 수 지정 (최대 6)
- 팀별 쿼터별 포메이션 선택 (모달)
- 팀별 쿼터별 포지션 배정
  - 선수 목록 → 필드 드래그 앤 드롭
  - 필드 내 포지션 교차 수정
  - 목록 ↔ 필드 간 교차 수정

### 공통 기능

- 포메이션별 포지션 랜덤 배치
- 이미지 저장 / 공유
- 실시간 동기화: 관리자가 경기 중 변경 시 전체 클라이언트 동기화 (WebSocket 또는 Polling)

## 경기 상세 페이지

### 헤더

- 경기 날짜 / 우리팀 vs 상대팀 / 결과 / 스코어

### 탑탭 네비게이터

#### 1. 경기 기록 탭

- 득점 기록 / 도움 기록
- 쿼터별 참여 선수 목록
- **MVP 투표하기 버튼** (경기 후 + 관리자가 기록 등록한 시점부터 활성화)

**MVP 투표 페이지**

- 활성화 조건: 관리자가 경기 기록 등록 → 푸시 알림 발송
- 마감: 경기 당일 자정
- 당일 MVP 지정 알림 전송
- 참여 선수 목록 (등번호 / 이름 / 포지션)
- 이름 검색 기능
- **중복 투표 방지**: DB `(userId, matchId)` Unique 제약
- **동점 처리**: 공동 MVP 처리 (복수 수상 가능)

#### 2. 댓글 탭

- 경기 댓글 작성 (글자 수 제한 필요)
- 댓글 목록 (작성자 / 작성일시 상대시간 / 좋아요)
- 동기화 전략 필요 / 푸시 알림 적용 여부 검토

#### 3. 영상 탭

- 유튜브 URL 입력 및 등록
- 유튜브 영상 재생 (웹뷰)

#### 4. 상대팀 평가 탭

> **매칭전에만 표시 — 자체전에서는 탭 숨김 처리**

- 우수 선수 선택하기
- 상대팀 평점 / 리뷰
- 평가 제출 버튼
- 상대팀 MVP 투표 (선택, 매너 점수 반영)

## 다가오는 / 오늘 경기 페이지

경기 상태(전·중·후)에 따라 UI 분기. 상태 전환은 시간 기반 자동.

### 경기 전

- D-Day 표시 (D-Day, D-3 강조)
- 경기 정보: 우리팀·상대팀 이름·레벨 / 날짜·시간 / 장소·주소
- 팀원 참여 현황: 참석 / 미정 / 불참 / 참석 명단
- 포메이션: 쿼터별 선택 / 각 포지션 / 관리자 수정 버튼
- 상대팀 정보: 팀명 / 레벨 / 평점 / 최근 5경기 전적

### 경기 중

> 상세 기획 추후 확정 (실시간 포지션 동기화 화면 연계 예정)

### 경기 후

**기록 입력 버튼 활성화 조건**

- 현재 시각 > 경기 종료 시각

**MVP 투표 활성화 조건**

- 관리자가 경기 기록 등록한 시점에 푸시 알림 발송
- 마감: 경기 당일 자정
- 당일 MVP 지정 알림 전송
