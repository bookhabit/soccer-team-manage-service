📌 설계 주요 결정사항
  • poll_votes.vote = 'yes'|'no'|'maybe' (3-state)
  • match_goals/assists에 guest_name 컬럼 → 용병 기록 지원
  • manner_scores는 집계 캐시 테이블 → match_evaluations 변경 시 트리거로 갱신
  • position_setups.positions_json → JSON 컬럼으로 포메이션 배정 저장
  • users.deleted_at → Soft Delete (탈퇴 처리)
  • polls ↔ matches = 1:0|1 (투표 없이 직접 기록 추가 가능)
